# 🚗 TireShopLoadPredictor

Интеллектуальная система прогнозирования сезонной нагрузки на сервисные центры шиномонтажа.

## 📌 О проекте

Веб-приложение, которое прогнозирует загрузку шиномонтажного сервиса на будущие периоды и определяет оптимальное количество рабочих боксов.

**Как работает:**
- Анализирует исторические данные Яндекс.Wordstat (2018–2024 гг.)
- Выявляет сезонные паттерны спроса с помощью нейросети PyTorch
- Рассчитывает метрики очереди (время ожидания, загрузка каналов) по модели M/M/n

## ⚙️ Технологии

| Категория | Инструменты |
|-----------|-------------|
| Язык | Python 3.11+ |
| ML / DL | PyTorch, scikit-learn, NumPy, Pandas |
| Web | FastAPI, Jinja2, Chart.js |
| Математика | Теория массового обслуживания (M/M/n) |

## 📊 Результаты

Тестирование на последних 12 месяцах данных (апрель 2025 – март 2026):

| Модель | MAE | MAPE | Точность |
|--------|-----|------|----------|
| **Нейросеть + сезонная коррекция** | **141,93** | **6,23%** | **93,77%** 🏆 |
| Нейросеть без сезонности | — | — | ~85% |
| Скользящее среднее | 959 | 33,23% | 66,77% |
| Линейная регрессия | 1211 | 75,21% | 24,79% |

**Вывод:** гибридный подход (нейросеть + экспертная сезонная коррекция) показал наилучшую точность, корректно улавливая пики спроса в октябре и апреле.

## 🚀 Быстрый старт

```bash
# Установка зависимостей
pip install -r requirements.txt

# Обучение модели
python scripts/train.py

# Запуск сервера
uvicorn app.main:app --reload
```

Открыть в браузере: `http://127.0.0.1:8000/`
